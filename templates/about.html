About the project

The web is the huge and most extravagant wellspring of data. To recover the information from the World Wide Web, Search Engines are commonly utilized. This paper covers different techniques which are used in recommendation system and also proposes a new system for efficient web page recommendation based on collaborative filtering i.e. using collaborative technique and CHARM algorithm which are coupled with the pattern discovery algorithm. Search engines provide a simple interface for searching for user query and displaying results in the form of the web address of the relevant web page but using traditional search engines has become very challenging to obtain suitable information. This paper proposed a search engine using Machine Learning technique that will give more relevant web pages at top for user queries.

CONCLUSION

Recently several recommendation systems have been proposed, that are based on collaborative filtering, content-based filtering, and hybrid recommendation techniques. Content-based filtering technique is one of the most successful recommendation techniques to solve the scalability problem related to recommendation systems and also providing better recommendations. In this paper, we cover all collaborative-based recommendation techniques which are used for better recommendations. This Hybrid recommendation method improves the prediction quality of this system.
